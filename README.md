## 🔷 Sorting Visualizer

A web-based application to visualize different sorting algorithms in action.

## 🚀 Live Demo
👉 https://123nitish.github.io/algo_visualization/

## 📌 Features
- Visualizes sorting algorithms step by step  
- Easy to understand the different sorting algorithm. 
- Helps in learning DSA concepts 
- how they can be used to solve the real-world problems.
   

## 🛠️ Tech Stack
- HTML  
- CSS  
- JavaScript(DoM manipulation,Event Handling)


## 📚 Algorithms Implemented
- Bubble Sort  
- Selection Sort  
- Insertion Sort  
- Merge sort
- Quick sort

## 🎯 Purpose
This project is built to understand how sorting algorithms work visually and improve problem-solving skills.
